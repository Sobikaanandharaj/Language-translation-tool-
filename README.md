
# 🌍 Language Translation Tool

A responsive Language Translation Tool built using HTML, CSS, and JavaScript. It translates text between multiple languages using a Translation API and includes features such as language selection, text-to-speech, and a clean, user-friendly interface.

## Features
- 🌐 Translate text between multiple languages
- 🔄 Swap source and target languages
- 🔊 Text-to-Speech (Listen)
- 📱 Responsive design
- ⚡ Fast API-based translation

## Technologies Used
- HTML5
- CSS3
- JavaScript
- Translation API

> Note: This project was developed with AI-assisted coding for learning, debugging, and implementation.
